**Description:** Demonstrates building a custom AWS Virtual Private Cloud (VPC) with public and private subnets, secure EC2 deployment, Bastion host access, and CloudWatch monitoring. This project showcases fundamental AWS networking concepts, secure infrastructure setup, and operational best practices.

---

**Objective:**
- Build a custom VPC (10.0.0.0/16) with one *public* and one *private* subnet.
- Set up *Internet Gateway* and *NAT Gateway* for proper routing.
- Launch *public and private EC2 instances* with appropriate security.
- Configure a *Bastion Host* for private EC2 access.
- Enable *CloudWatch monitoring* for the instances.
